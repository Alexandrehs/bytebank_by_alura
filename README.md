# bytebank

App desenvolvido em flutter.

## Sobre
O app faz parte do curso de flutter da Alura, no app pode se criar uma lista de contatos, 
os dados são persistidos no aparelho, assim fica salvo mesmo após fechar o app.

## Tecnologias
Flutter || Sqflite
